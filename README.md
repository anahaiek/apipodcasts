# 📻 Podcast API — Node.js, TypeScript & Native HTTP Module

API REST desenvolvida com Node.js e TypeScript, utilizando exclusivamente o módulo HTTP nativo do Node.

A aplicação disponibiliza um endpoint GET responsável por retornar uma lista de podcasts a partir de uma fonte de dados em JSON, simulando um cenário real de consumo de dados.

## ✨ Principais características
Implementação de API utilizando HTTP nativo do Node.js
Código totalmente tipado com TypeScript
Estrutura organizada em Controller, Model e Repository
Endpoint GET para listagem de podcasts
Leitura de dados a partir de arquivo JSON
Configuração de variáveis de ambiente com dotenv
Projeto leve, performático e de fácil manutenção
## 📡 Endpoint disponível
#### http://localhost:3636/api/episode
#### http://localhost:3636/api/episode/list
#### http://localhost:3636/api/episode?p=flow
#### http://localhost:3636/api/episode?p=venus
Retorna uma lista de podcasts em formato JSON.
